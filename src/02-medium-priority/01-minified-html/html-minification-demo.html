üì¶ HTML Minification Example

HTML minification removes unnecessary characters from HTML code without changing its functionality. This includes removing whitespace, comments, and line breaks.

‚ùå Before Minification (Original HTML)

<!DOCTYPE html>
<html lang="en">
  <head>
    <!-- Page metadata -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website</title>
    
    <!-- Styles -->
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <!-- Main navigation -->
    <nav class="navigation">
      <ul>
        <li><a href="/">Home</a></li>
        <li><a href="/about">About</a></li>
        <li><a href="/contact">Contact</a></li>
      </ul>
    </nav>
    
    <!-- Main content -->
    <main>
      <h1>Welcome to My Website</h1>
      <p>
        This is a simple example of HTML
        that contains lots of whitespace
        and comments for readability.
      </p>
    </main>
    
    <!-- Footer section -->
    <footer>
      <p>© 2025 My Website</p>
    </footer>
    
    <script src="app.js"></script>
  </body>
</html>

üöÄ Benefits of HTML Minification

  • Reduced File Size: Smaller HTML files load faster
  • Faster Page Load: Less data to download and parse
  • Lower Bandwidth: Saves data for users and reduces server costs
  • Better Performance: Improves overall site performance metrics
  • SEO Benefits: Faster loading pages rank better in search engines

const minify = require('html-minifier').minify;
const result = minify('<p>    Hello World!    </p>', {
  removeComments: true,
  collapseWhitespace: true,
  minifyJS: true,
  minifyCSS: true
});
