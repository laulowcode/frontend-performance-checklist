What to look for

  • Separate mode: network shows two CSS requests: styles-a.css and styles-b.css.
  • Concat mode: network shows one CSS request: styles-concat.css.
  • With HTTP/1, fewer requests can reduce overhead. With HTTP/2+, test before deciding.
