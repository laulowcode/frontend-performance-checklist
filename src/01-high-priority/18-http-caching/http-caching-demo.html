HTTP Caching Demo

❌ No Caching

Cache-Control: no-cache, no-store

Every request goes to server, no performance benefit

✅ Proper HTTP Caching

Cache-Control: public, max-age=31536000, immutable

Files cached for 1 year, massive performance boost

Static Assets (CSS/JS/Images)

Headers:
Cache-Control: public, max-age=31536000, immutable
ETag: "abc123"

Cache for 1 year with versioning

HTML Documents

Headers:
Cache-Control: no-cache, private
ETag: "def456"

Always validate, cache in browser only

Cache Control Examples

// Static assets with versioning Cache-Control: public, max-age=31536000, immutable // HTML documents Cache-Control: no-cache, private // User-specific content Cache-Control: private, max-age=3600 // Sensitive data Cache-Control: no-store

ETag Validation

// First request GET /style.css Response: 200 OK, ETag: "abc123" // Second request (cached) GET /style.css If-None-Match: "abc123" Response: 304 Not Modified (saves bandwidth!)
